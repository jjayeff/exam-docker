# Proxumer - Docker

## Quick Start

```bash
1. git clone https://github.com/jjayeff/exam-docker.git

2. docker-compose up -d

3. go to play chat (http://localhost)
# frontend http://localhost
# backend http://localhost:4000

**don't forget [docker-compose down] when you done!!

# Run in Docker
docker-compose up
# use -d flag to run in background

# Tear down
docker-compose down

# To re-build
docker-compose build
```
